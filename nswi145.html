    • Eclipse IDE for Java EE Developers, at least Oxygen 3
    • SoapUI
    • One of:
      • Java JDK 8, at least 1.8.0_u201

  7. WS-BPEL. Prerequisites:
    • Apache ODE 1.3.8 WAR distribution
