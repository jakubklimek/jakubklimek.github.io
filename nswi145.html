NSWI145 - Web Services

Basic information - summer 2021

  1. The lectures and tutorials will be based on asynchronous self-study using the materials provided on this webpage.
  2. The lecture slides contain speaker notes with what is typically said on the lecture.
  3. Each tutorial has a series of tasks to be tried out. Again, note the speaker notes identifying common pitfalls and how to avoid them.
  4. Each tutorial has a homework assignment to be completed and sent to me via email with [NSWI145] in subject before the next tutorial!
  5. The problems identified by me in the homework are to be corrected and resent with the next assignment.
  6. You can do the final exam if and only if you did all 10 homework assignments successfully.
  7. Online consultations regarding each week's topic can be arranged in the time slot of the tutorial i.e. Mondays 10:40-12:10 CET. Write me an email first.

Slides - labs

Warning: the examples in the tutorials do not work with JDK 16 at the moment.

  1. 2021-03-01: Prerequisites tutorial (HTTP, XML, tools). Prerequisites:
    • Visual Studio Code as a handy text editor
    • For Windows users: PuTTY as a Telnet client
    • For Windows users: Curl
    • Postman
  2. 2021-03-08: SOAP 1. Prerequisites:
    • Eclipse IDE for Java EE Developers
    • SoapUI
    • One of:
      • Newest OpenJDK and Maven dependencies for JAX-WS
      • Newest OpenJDK (e.g. OpenJDK 15) and libraries
        • JAX-WS RI 2.3.0
        • javax.activation
      • Java JDK 8, at least 1.8.0_u281
  3. 2021-03-15: WSDL
  4. 2021-03-22: WSDL+SOAP, SAAJ
    • Apache Tomcat 9
  5. 2021-03-29: CXF, WS-*. Prerequisites:
    • Apache Tomcat 9
    • Apache CXF 3.4.x
  6. 2021-04-12: UDDI. Prerequisites:
    • Apache jUDDI 3.3.9
    • Apache Derby 10.10.1.1
    • In Eclipse, Import as existing Maven project juddi-distro-3.3.9/examples/uddi-annotations - involves lots of downloading
  7. 2021-04-19: WS-BPEL. Prerequisites:
    • Apache ODE 1.3.8 WAR distribution
    • In Eclipse (for Java EE), install BPEL Commons, BPEL Runtime Adapter for Apache ODE 1.3, BPEL Visual Designer
  8. 2021-04-26: REST. Prerequisites:
    • Jersey 3.x bundle
    • Apache Tomcat 10
  9. 2021-05-03: LOD introduction
  10. 2021-05-10: RDF, RDFS, Serializations
  11. 2021-05-17: SPARQL
  12. 2021-05-24: Solid
  13. 2021-05-31: No tutorial
